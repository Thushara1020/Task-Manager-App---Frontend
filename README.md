# 🚀 Task Manager Application

A professional Full-Stack Task Management System designed for efficient task tracking and team collaboration.

---

## 🛠 Tech Stack

### **Frontend**
- **Framework:** Angular v21.1.4
- **Styling:** Tailwind CSS (Modern Glassmorphism UI)
- **Testing:** Vitest

### **Backend**
- **Framework:** Java Spring Boot
- **Database:** MySQL

---

## 📂 Project Structure

- **`/frontend`**: Angular source code including components for task forms and dashboards.
- **`/backend`**: Spring Boot REST API for managing users, tasks, and data persistence.
